# sub-FSM Supplement

This folder contains supplemental files for the ROS voice-interaction ordering sub-FSM.

Included files:
- `main.py`: overall voice-interaction startup entry
- `bridge/ros_voice_bridge.py`: ordering sub-FSM core logic
- `bridge/__init__.py`: bridge package init
- `scripts/test_order_subfsm_stability.py`: offline sub-FSM stability test
